- [Java](/language/javase)

> 解释型语言与编译型语言区别：
   - 在程序执行之前，通过编译链接等过程，将程序翻译成机器语言，运行时，无需翻译直接执行。（与底层平台紧密结合，可移植性差）
   - 解释型语言由解释器解释，在运行期翻译成机器语言，相比编译型语言，性能更低，但是，通常可移植性较好。（脚本语言属于解释型语言）
